At Publicis Sapient, my team and I created a chatbot which our client's senior leaders could use to gain insights on their operations data, enabling better business decisions.

React: Developed the interactive frontend interface.
Node.js: Implemented the backend server for handling API requests.
Google Cloud Run: Deployed and managed our scalable containerized application.
Vertex AI: Utilized the Gemini 1.5 Flash model for machine learning predictions.
Langchain: Natural language processing and prompt engineering to refine model.
BigQuery: Performed large-scale operations data analysis and querying.
Python: Developed AI model workflows and integrated with BigQuery.
Flask: Created a lightweight web server to interface Python logic with the frontend.
